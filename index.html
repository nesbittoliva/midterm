Character Info Generator

This project is a simple PHP website that generates a movie character profile based on user input. CometeAlgo Calorie Tracker

CometeAlgo is a simple calorie tracker built with HTML, CSS, and JavaScript. Users can input food items along with their calorie values, view a running list of entries, and see a total calorie count update in real time. This project is a work in progress — additional features like item deletion, data persistence, and custom styling are planned in future updates.
